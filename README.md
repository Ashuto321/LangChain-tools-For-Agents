<h1 align="center">LangChain Tools & Agent Development</h1>

<p align="center">
A comprehensive hands-on project demonstrating how to build, integrate, and execute 
<strong>Custom Tools</strong> and <strong>Built-in Tools</strong> within LangChain Agents.
This repository explores tool creation, tool calling, tool execution, and agent-driven decision making
to build intelligent AI systems capable of interacting with external APIs and performing real-world tasks.
</p>


<h2>Project Overview</h2>

<p>
This repository focuses on the practical implementation of LangChain Tools and Agents, covering the complete workflow from tool creation to agent execution. The project demonstrates how Large Language Models (LLMs) can dynamically select and utilize tools to solve user queries efficiently.
</p>

<p>
Alongside core LangChain concepts, a real-time <strong>Currency Converter Tool</strong> has been developed using an Exchange Rate API, showcasing how external services can be integrated into AI-powered applications.
</p>

<h2>Key Features</h2>

<ul>
    <li>Development of Custom LangChain Tools using the <code>@tool</code> decorator</li>
    <li>Implementation of Built-in LangChain Tools</li>
    <li>Tool Calling and Automatic Tool Selection by Agents</li>
    <li>Tool Execution Workflow and Response Handling</li>
    <li>External API Integration for Real-Time Data Retrieval</li>
    <li>Currency Conversion Tool using Exchange Rate API</li>
    <li>Agent-Based Decision Making and Reasoning</li>
    <li>Production-Oriented Examples with Clean Code Structure</li>
</ul>

<h2>Technologies Used</h2>

<ul>
    <li>Python</li>
    <li>LangChain</li>
    <li>LangChain Agents</li>
    <li>LLMs (OpenAI / Groq Compatible Models)</li>
    <li>REST APIs</li>
    <li>Exchange Rate API</li>
</ul>

<h2>Learning Outcomes</h2>

<p>
Through this project, I gained practical experience in designing AI agents that can reason, select appropriate tools, interact with external systems, and return accurate responses. The repository serves as a strong foundation for building advanced applications such as AI Assistants, RAG Systems, Automation Workflows, and Agentic AI Solutions.

<p>
Understanding tool usage is a critical skill in modern Generative AI development. This repository demonstrates the core concepts behind agentic systems and showcases how LLMs can be enhanced with external capabilities to solve complex real-world problems beyond text generation.
